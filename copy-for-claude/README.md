A VS Code extension for copying highlighted code reference to the clipboard to be pasted into a Claude Code prompt.

Install by running the "Developer: Install Extension from Location" command in VS Code.

Highlight a single or multiple lines, then press `Option + K`.

Example outputs:
* @pro-iq/lib/riverpod/video_repository.dart#9-19
* @pro-iq/lib/riverpod/providers.dart#24
